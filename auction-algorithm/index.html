Algoritmo de Leilão

Lógica do algoritmo


  • Cada origem é vista como como pessoa i, e cada destino como o objeto j em um leilão.

  • Cada pessoa i possui um interesse aij por um objeto j, que inicialmente tem um preço pj.

  • O valor real de um objeto j para uma pessoa i é:
    vr = aij- pj


Satisfação e condição de parada


  • Uma pessoa estará satisfeita se ela sempre ficar alocada ao objeto que traga o maior valor real:
    vr = max {aij– pj} | j = {1, 2, ... n}

  • Objetivo é otimizar as atribuições, considerando o custo/benefício de cada relação (pessoa → objeto).
